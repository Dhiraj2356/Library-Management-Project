# Library-Management-Project
SQL based library management system with relational tables and data analysis

##  Project Overview
This project is a relational **Library Management System** designed using SQL.  
The objective is to manage and analyze library operations such as book issuance, returns, member activity, and employee management by using well-structured tables, foreign key relationships, and SQL queries.

The project demonstrates practical database design, data manipulation, and analytical querying skills.

---

##  Objectives
- Design a relational database for a library system  
- Establish relationships between tables using primary and foreign keys  
- Perform CRUD operations to manage library data  
- Answer business-related questions using SQL queries  
- Apply advanced SQL concepts for data analysis  

---

##   CRUD Operations
- Inserted sample data across all tables  
- Performed **CRUD operations**:
  - **Create**: Added new books, members, and transactions  
  - **Read**: Retrieved records using SELECT queries  
  - **Update**: Modified existing records such as book status  
  - **Delete**: Removed outdated or incorrect entries
